Add customer

Add a customer to the database. The customer has a first name (VNaam), a family name (FNaam), a street (Straat), a number (Nr), a postcode (Postcode) and a city (Gemeente). The customer is added to the database when the form is submitted. Every input that has a name, will be sent to the server when the form is submitted. In Python you can access the values of the inputs with the request.form dictionary.
